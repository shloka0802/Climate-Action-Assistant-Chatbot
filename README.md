# Climate Action Assistant Chatbot

SDG
SDG 13 – Climate Action

Project Description
This project is a Climate Action Assistant Chatbot developed using Botpress. The chatbot helps users learn about climate change, energy conservation, water conservation, plastic waste reduction, and environmental protection.

Features
- Climate change awareness
- Energy saving tips
- Water conservation guidance
- Plastic waste reduction tips
- Environmental protection awareness
- Fallback response for unrelated questions

Technology Stack
- Botpress
- Artificial Intelligence (AI)
- Natural Language Processing (NLP)
- GitHub
